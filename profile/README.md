Netrunner UG is a small side-gig doing all kinds of embedded and IoT development and research, as well as security related stuff that aligns with it. The main project right now is an open source KNX device stack that is as conformance tested as possible without dipping our feet too deep into the KNX Association and a required membership.

## Projects

Here is a current list of the biggest projects we actively work on right now:

* [zweidraehte](https://github.com/ntrnnr/zweidraehte), our open source KNX device stack for everybody to use.

## Work with us

If you want to help improving any of these projects, feel free to submit a PR with your changes and we'll go from there.

If you need consulting around general KNX protocol work, security reviews, design help or require special licensing (at least the KNX stack has the possibility for commercial closed-source licensing) for one of our projects, feel free to reach out info@netrunner.info and we'll get back to you ASAP.
